# 📑 Инструкция по добавлению собственных фотографий проекта SREDA

Поскольку вы сгенерировали новые красивые фотографии для вашего портфолио в формате **PNG**, этот гайд поможет вам правильно разместить их в проекте и подключить на сайт.

---

## 📂 Шаг 1. Куда загружать файлы

В проектах на Vite самый простой способ работы со статическими изображениями — использовать специальную папку `public` в корне проекта. Все файлы из этой папки будут доступны на сайте напрямую по их имени.

1. Если в корне проекта (рядом с `package.json`, `src` и `index.html`) ещё нет папки `public`, **создайте её**.
2. Поместите ваши сгенерированные файлы `.png` внутрь созданной папки `public`.

---

## 🏷️ Шаг 2. Как назвать файлы (рекомендация)

Чтобы не запутаться, назовите файлы структурировано в соответствии с каждым проектом (До / После).

## 🛠️ Шаг 3. Как подключить их в коде сайта

Откройте файл `/src/components/Portfolio.tsx`. В самом начале файла вы увидите массив проектов под названием `PORTFOLIO_ITEMS`. 

Вам нужно заменить существующие ссылки `https://images.unsplash.com/...` на пути к вашим локальным файлам, начиная со слэша `/`.

### 📝 Пример замены для Проекта 2 (Кухня на Патриарших):

**Было:**
```typescript
{
  id: 'proj-2',
  title: 'Кухня-гостиная на Патриарших',
  // ...
  beforeImage: 'https://images.unsplash.com/photo-1560185127-6a2806647f81?auto=format&fit=crop&w=800&q=80',
  afterImage: 'https://images.unsplash.com/photo-1556911220-e15b29be8c8f?auto=format&fit=crop&w=800&q=80',
  // ...
}
```

**Стало (после того как вы загрузили файлы в `public`):**
```typescript
{
  id: 'proj-2',
  title: 'Кухня-гостиная на Патриарших',
  // ...
  beforeImage: '/patriarshie-before.png',
  afterImage: '/patriarshie-after.png',
  // ...
}
```

## 💡 Полезный совет по оптимизации PNG
Сгенерированные нейросетями PNG файлы могут весить очень много (по 5-10 МБ каждое). Чтобы сайт загружался мгновенно, перед загрузкой рекомендуется сжать их через любой бесплатный сервис (например, **TinyPNG** или аналогичный компрессор). Это уменьшит размер картинок на 70-80% без видимой потери качества!
